# ART20 Token Standard

ART20 is the native token interface used within the Arta Blockchain ecosystem.

The standard defines the basic functionality required for interoperable tokens including transfer logic, approvals, and balance tracking.

## Core Functions

- totalSupply()
- balanceOf(address)
- transfer(address,uint256)
- approve(address,uint256)
- allowance(address,address)
- transferFrom(address,address,uint256)

## Purpose

The ART20 standard is designed to provide compatibility with DeFi infrastructure, exchanges, and cross-chain integrations.

This specification serves as the reference implementation for tokens deployed within the Arta Blockchain ecosystem.
