# Contributing to Arta Blockchain

We welcome contributions from developers and researchers.

## Development Guidelines

• Follow Solidity best practices  
• Keep contracts modular and well documented  
• Write clear commit messages  
• Include tests for new functionality

## Pull Requests

1. Fork the repository
2. Create a feature branch
3. Submit a pull request with description of changes

All submissions will be reviewed by the Arta Blockchain development team.
