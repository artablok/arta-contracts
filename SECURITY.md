# Security Policy

The Arta Blockchain team takes security seriously.

If you discover a vulnerability within the smart contracts or infrastructure of the Arta ecosystem, please report it responsibly.

## Reporting a Vulnerability

Please send a detailed report to:

security@artablockchain.in

Include:

• description of the vulnerability  
• steps to reproduce  
• potential impact  
• suggested mitigation if available

We will review all reports and respond as quickly as possible.

Responsible disclosure is appreciated.
